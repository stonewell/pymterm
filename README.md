# pymterm
A multiple tab terminal emulator implemented in python

UI framework 
 - https://kivy.org
 - https://github.com/stonewell/pygui/

ssh library https://github.com/paramiko/paramiko

## Dependecy
 - paramiko
 - appdirs
 - pypiwin32/pygtk/pycoco
 - pyopengl pyopengl-accelerate
 - pygame
 - numpy
 - functools32
